Praman — Proof That Demands Action

Turning scattered civic complaints into collective evidence and public accountability

Problem

In India, civic issues like broken roads, garbage overflow, and unsafe public spaces are:

frequently experienced
occasionally reported
rarely resolved

Current systems fail because:

complaints are isolated and ignored
there is no aggregation of proof
there is no visibility of inaction

One complaint is easy to dismiss.
There is no system that turns repeated complaints into undeniable evidence.

Solution

Praman transforms:

Individual complaints → Collective proof → Public pressure

Instead of relying on single reports, it builds consensus-driven cases and makes inaction visible.

How It Works
Report
Upload geo-tagged image of an issue
Validate
Others confirm or report the same issue
Aggregate
Similar reports form a Case
Escalate by Consensus
5 reports → Emerging
10 reports → Verified
15+ reports → Public Case
Public Pressure Board
Displays:
    issue proofs
    number of reports
    “No action taken for X days”

Key Features
Geo-tagged issue reporting
Smart case aggregation
Escalation by consensus
Public Pressure Board
Inaction tracking
Proof Strength Indicator
Shareable case links
Optional complaint generator

Impact
Encourages collective civic participation
Makes inaction visible and measurable
Builds pressure without confrontation

Praman doesn’t just collect complaints — it exposes how long they are ignored.
